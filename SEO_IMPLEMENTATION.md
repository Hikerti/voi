# SEO и семантическая структура Voitov Studio

## Что реализовано

- Базовое семантическое ядро в `lib/seo.ts`.
- Единый генератор metadata: Title, Description, Keywords, Canonical, Robots, Open Graph и Twitter Card.
- JSON-LD для организации и сайта.
- JSON-LD для услуг, статей, новостей, FAQ и проектов.
- Один смысловой H1 на основных страницах и последовательные H2–H6 в контенте.
- Общие стили заголовков, списков, абзацев, ссылок, strong/b и цитат.
- Хлебные крошки и `BreadcrumbList` JSON-LD.
- Sitemap и robots.txt.
- Основной домен `https://voitov.ru`, единый trailing slash и редиректы старых адресов.
- Статический поиск по услугам, работам, статьям, новостям и FAQ.
- SEO-текст на главной и страницах услуг.
- Open Graph placeholder `public/images/og-cover.svg`.

## Временное семантическое ядро

Пока не получено утверждённое ядро, используются группы:

1. Разработка сайтов
   - разработка сайтов;
   - создание сайтов в Москве;
   - заказать сайт;
   - корпоративный сайт;
   - лендинг.
2. Дизайн
   - веб-дизайн;
   - UI/UX дизайн;
   - создание логотипа;
   - брендинг.
3. SEO и маркетинг
   - SEO-продвижение;
   - SEO-подготовка сайта;
   - контентная стратегия;
   - digital-маркетинг.
4. Поддержка
   - поддержка сайта;
   - развитие сайта;
   - сопровождение сайта.

## Что заменить после получения данных заказчика

- юридическое название, ИНН, ОГРН, адрес и график;
- второй телефон и реальные ссылки соцсетей;
- Яндекс Карту;
- утверждённые Title, Description и H1;
- поисковые запросы и региональность;
- тексты услуг, цены и преимущества;
- юридический текст политики конфиденциальности;
- placeholder Open Graph на утверждённый JPG/PNG;
- временные SEO-блоки на тексты под реальное семантическое ядро.

## Правила разметки

- На индексируемой странице должен быть один основной H1.
- H2 — крупные смысловые разделы страницы.
- H3 — подразделы внутри H2.
- H4–H6 используются только при реальной вложенности, а не ради размера текста.
- Текстовые ссылки подчёркиваются; декоративное подчёркивание для обычного текста не используется.
- Списки оформляются через `ul/ol/li`, а не через переносы строк.
- Изображения получают содержательный `alt`, декоративные изображения — пустой `alt`.
- Текущий пункт меню имеет `aria-current="page"` и не является ссылкой.
- Utility-страницы поиска, спасибо и 404 закрыты от индексации.

## Проверка после деплоя

```bash
curl -I https://voitov.ru/
curl -I https://www.voitov.ru/
curl -I https://voitov.ru/index.php
curl -I https://voitov.ru/services
curl https://voitov.ru/robots.txt
curl https://voitov.ru/sitemap.xml
```

Дополнительно проверить исходный HTML основных страниц: `title`, `description`, `canonical`, Open Graph, JSON-LD и количество H1.
